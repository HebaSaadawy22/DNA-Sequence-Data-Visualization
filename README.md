# DNA Sequence Data Visualization (Bioinformatics) 🧬

This project analyzes and visualizes DNA sequences from three different species: Human, Chimpanzee, and Dog, using Python for data preparation and Google Looker Studio for interactive dashboarding.

## 🛠️ How it Works
1. **Data Preparation**: Combined and cleaned the genomic data files using Python (`pandas`) to handle biological sequences.
2. **Feature Engineering**: Calculated the total sequence length for each gene sequence.
3. **Data Visualization**: Designed an interactive dashboard using Google Looker Studio to track:
   * **Average Sequence Length** per organism (Dog, Chimpanzee, and Human).
   * **Distribution of Gene Classes** (Families coded from 0 to 6).
